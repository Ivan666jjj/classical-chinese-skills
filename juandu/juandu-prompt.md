# 📖 古籍句读

自动为古籍添加标点断句。基于古代汉语语法特征和语气词规律。

## 断句原则（依王力体系）
| 标志 | 常断句 |
|---|---|
| 者、也、矣、焉、哉 | 语气词后断 |
| 乎、耶、欤 | 疑问句末断 |
| 曰、云、谓 | 对话前后断 |
| 然则、虽然、且夫 | 发语词前断 |
| 故、是故、是以 | 因果句前断 |

## 使用
```
/juandu 子曰学而时习之不亦说乎有朋自远方来不亦乐乎
→ 子曰：「学而时习之，不亦说乎？有朋自远方来，不亦乐乎？」
```
