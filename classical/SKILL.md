# 📜 半文半白写作 — Classical Literary Style

以正史笔法为本（史记、资治通鉴、二十四史），半文半白，自然雅驯。

## 语法规范
| 规则 | 示例 |
|---|---|
| 用「之」代「的」 | 日落之时 → 日落的时候 |
| 用「其」代「他的/它的」 | 其色如丹 → 它的颜色像红色 |
| 用「者」表停顿 | 云厚者，光不透也 |
| 用「所」构成所字结构 | 所见者，晚霞漫天 |
| 用「于」代「在/对」 | 于福海东岸观日落 |
| 用「而」连接 | 云散而日现 |
| 用「则」表承接 | 雨过则天晴 |
| 用「乃」表判断 | 此乃火烧云也 |
| 用「然」表转折 | 云厚，然西隅有光 |
| 用「矣/也/乎/哉」收尾 | 可往观矣 / 不亦乐乎 |

## 行文特点
- 短句为主，叙事简洁（效史记笔法）
- 保留现代词汇（电脑、手机、软件等）
- 禁用 AI 味浓词（"深入探究""绚丽画卷"等）
- 结论明确，不加虚词堆砌

## 使用
```
/classical 帮我写一段今天看日落的记录
→ 今日暮时，余至圆明园福海东岸。
  云厚而西隅透光，俄而色转绯红，天地尽染。
  此数日来难得之景，不虚此行也。
```

> 语法参照《史记》《资治通鉴》及民国学人半文半白文体
