# 📜 半文半白写作 — 王力《古代汉语》语法体系

依王力先生《古代汉语》四卷本语法框架。半文半白，自然雅驯。

## 虚词体系（王力《古代汉语》第一册·通论）

| 虚词 | 用法 | 示例 |
|---|---|---|
| 之 | 连词（定中之间）/ 代词 | 日落之时 / 观之甚美 |
| 其 | 人称代词 / 指示代词 | 其色绯红 / 其一为火烧云 |
| 者 | 辅助性代词（……的人/物） | 云厚者，光不透也 |
| 所 | 所字结构（所+动=名） | 所见者，晚霞漫天 |
| 于 | 介词（引进行为对象处所） | 于福海东岸观日落 |
| 而 | 连词（并列/顺承/转折） | 云散而日现 / 厚而无光 |
| 则 | 连词（承接/因果/假设） | 雨过则天晴 |
| 乃 | 副词（表判断/确认/于是） | 此乃火烧云也 |
| 然 | 连词（表转折）/ 应答 | 云厚，然西隅有光 |
| 矣/也/乎/哉/焉 | 语气词 | 可往观矣 / 不亦乐乎 |

## 词类活用（王力《古代汉语》第二册·通论）
- 名词用作动词：左右欲兵之 / 范增数目项王
- 使动用法：小天下（使天下变小）
- 意动用法：奇之（认为……奇特）
- 为动用法：死之（为……而死）

## 特殊句式（王力《古代汉语》第二册）
- 判断句：「……者，……也」
- 被动句：「为……所……」
- 否定句：不 / 弗 / 勿 / 毋 / 未 / 莫
- 反问句：「岂……乎」「安……哉」「何……为」

## 行文规范
- 短句为主，效《史记》叙事法
- 保留现代词汇（电脑、手机、软件等）
- 禁用 AI 高频词（"深入探究""绚丽画卷"）
- 名词前不加「的」字时用「之」
- 现代概念用现代词，不强行仿古
