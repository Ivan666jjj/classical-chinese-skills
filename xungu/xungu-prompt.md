# 📚 训诂对照

古书注解与训诂分析。依传统训诂学方法。

## 训诂方法（依黄侃、王力体系）
| 方法 | 说明 |
|---|---|
| 形训 | 以字形说解字义（依《说文》） |
| 声训 | 以音近推求词源（依《释名》） |
| 义训 | 以通行语义解释 |
| 互文见义 | 上下文对照求义 |
| 比较互证 | 同义/反义词系联 |

## 常用训诂术语
| 术语 | 含义 |
|---|---|
| 某，某也 | A就是B |
| 某犹某也 | A相当于B |
| 某谓之某 | A叫做B |
| 某之言某也 | A是B的声训 |

> 依《尔雅》《释名》《说文》及黄侃、王力训诂学体系
