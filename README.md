## 🚀 零基础安装指南

### 方式一：用 Reasonix（推荐，功能最全）

> **Reasonix** 是一个集成 AI 助手的桌面应用，本仓库所有技能都在上面开发测试。

**① 下载 Reasonix**
- 官网下载：https://reasonix.ai（Mac / Windows 都支持）
- 或者直接在应用商店搜「Reasonix」

**② 安装技能（三选一）**

方法 A — 一键安装（最简单）：
```bash
/smart-weather 帮我安装 classical-chinese-skills 全部技能
```
AI 会自动帮你装好。

方法 B — 手动复制：
打开这个链接 → https://github.com/Ivan666jjj/classical-chinese-skills
进入每个技能文件夹（如 `hanzi/`）→ 打开 `SKILL.md` → 全选复制
回到 Reasonix → 粘贴到对话框中即可。

方法 C — 直接输入命令（装好后）：
```bash
/hanzi 日              # 查说文解字
/yinyun 白日依山尽     # 测平仄
/wenxian 天行健        # 查经学引证
/classical 写一段日记   # 半文半白写作
```

---

### 方式二：用 Kimi / ChatGPT / Claude / DeepSeek

> **不需要安装任何软件，打开网页就能用。**

**Kimi**（国内免费，推荐新手）：
① 打开 https://kimi.moonshot.cn
② 复制本仓库任意 `SKILL.md` 文件的内容（比如 `hanzi/SKILL.md`）
③ 粘贴到 Kimi 对话框里，发送
④ 之后就可以问：「查一下「日」字的说文解字」

**ChatGPT**（需要科学上网）：
① 打开 https://chat.openai.com
② 点击左下角头像 → Custom Instructions
③ 把 `SKILL.md` 的内容粘贴进去
④ 保存后即可使用

**Claude / DeepSeek / 豆包 / 通义千问**：
操作方式同上——复制 `SKILL.md` 内容，粘贴到对话框或自定义指令中。

---

### 方式三：浏览器插件（Edge / Chrome）

① 打开 Edge 浏览器，地址栏输入 `edge://extensions`
② 打开右上角「开发人员模式」
③ 下载本仓库的 `browser-extension` 文件夹
④ 点「加载解压缩的扩展」→ 选择下载好的文件夹
⑤ 安装成功！右上角出现 📜 图标

之后在网页上选中文字 → 右键 → 「汉语言查询」即可。
