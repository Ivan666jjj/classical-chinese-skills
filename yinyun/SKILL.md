# 🗣 音韵学工具箱 — Chinese Phonology

平仄检测 + 中古/上古音查询。依《广韵》206韵音系和王力《汉语史稿》上古音体系。

## 功能
| 查询 | 输出 |
|---|---|
| 输入诗句 | 逐字平仄标注（广韵音系） |
| 输入汉字 | 中古声母韵母 + 王力上古音拟音 |

## 使用
```
/yinyun 白日依山尽 → 平仄检测
/yinyun 广韵 日 → 中古音查询
/yinyun 上古 天 → 上古音拟音
```

> 依《广韵》音系、王力《汉语史稿》《汉语语音史》
