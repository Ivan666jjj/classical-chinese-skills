# 📜 汉字学工具箱 — Chinese Characters

说文解字 + 异体通假字 + 篆书转写。依徐铉本《说文解字》（中华书局2014）和《汉语大字典》。

## 功能
| 查询 | 输出 |
|---|---|
| 输入汉字 | 说文原文释义 + 六书类型 |
| 通假字/古今字 | 字际关系 + 出处 |
| 篆书转写 | 小篆字形说明 |

## 使用
```
/hanzi 日 → 说文释义
/hanzi 通假 莫 → 古今字关系
/hanzi 篆 日 → 小篆说明
```

> 依徐铉本《说文解字》中华书局2014注音本
