# 📐 诗词格律检测

检测诗词平仄格律。依王力《诗词格律》体系，自动标注平仄、指出出律处。

## 功能
| 输入 | 输出 |
|---|---|
| 五言/七言诗句 | 平仄标注 + 格律分析 + 拗救建议 |
| 完整律诗/绝句 | 逐句平仄 + 押韵检查 + 对仗分析 |

## 格律框架（王力体系）
- 五言律句：仄仄平平仄 / 平平仄仄平 / 仄平平仄仄 / 平仄仄平平
- 七言律句：平平仄仄平平仄 / 仄仄平平仄仄平
- 拗救规则：本句自救 / 对句相救
